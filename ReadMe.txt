To Run the Project.
Option 1:
    1. Find the index.html file in the Project Folder.
    2. Right-click on index.html
    3. Run on Morilla Firefox (tested with Firefox Quantum v60.0.1).
Option 2:
    1. Open the Project folder in Brackets (tested with v1.12).
    2. Make sure the all project files are loaded properly.
    3. Open the index.html file and Click Run.
Option 3:
